
# Sky Ace v2

### Controls
- Arrow Keys: Fly (Up = Thrust, Down = Brake, Left/Right = Turn)
- Space: Fire bullets
- V + Space: Overdrive fire (40/sec)
- M: Fire homing missile
- B: Drop bombs (ground targets only)
